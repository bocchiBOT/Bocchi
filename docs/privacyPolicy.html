Privacy Policy

1. Data Collected

  • Some data is collected and stored to enable specific features of the application.
  • Among the data collected and stored are server IDs, channel IDs, role IDs, and messages generated by the application.
  • All collected and stored IDs are obtained via the Discord platform and are already accessible to any user with a developer account on the platform.
  • No other data, besides the IDs listed above, or sensitive data is collected.

2. Use of Data

  • Collected data is used solely for the functioning of the app’s features.
  • Data is not shared with third parties or publicly exposed.

3. User Control

  • Systems or commands that store IDs include commands, buttons, or additional functionalities that allow the user to disable the feature. When used, the feature is deactivated, and all related data is completely removed, without any backup or additional persistent storage.
  • Only the application owner(s) have access to this data.

4. Data Security

  • Reasonable measures are taken to protect stored data; however, the application owner(s) are not responsible for data loss, unauthorized disclosure, or issues caused by technical failures, the Discord platform, or events beyond the owner(s)’ control.

5. Policy Changes

  • This policy may be updated at any time without prior notice by the application owner(s). Continued use of the app constitutes acceptance of the latest version of the policy. It is recommended that users periodically review updates.
